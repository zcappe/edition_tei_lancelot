# Edition du Lancelot en prose

Edition à visée paléographique du *Lancelot en prose* en XML-TEI réalisée par Zoé Cappe dans un cadre pédagogique.


Dans le dossier **Edition** se trouvent:

1. La *DTD* de l'édition
2. L'encodage du texte en *XML-TEI*
3. La versioni *XHTML* de l'*ODD*
4. L'*ODD* de l'édition
5. Le *RelaxNG* de l'édition


Dans le dossir **Images** se trouvent, au format *JPEG*:

- le f.101v
- le f.102r
- le f.102v

du roman, qui font l'objet de la présente édition.


*Mise en ligne en février 2022*
